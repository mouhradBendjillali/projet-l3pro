La fonction ci-dessous se charge de copier le texte contenu dans le text input $( "textInput" ) directement dans le text area $( "textArea" ) au momet du clique sur le bouton Click me $( "copyText" ).

La fonction ci-dessous se charge d'instancier un slider dans la div $( "#slider-range-min" ), dont la valeur minimale est 0, et la valeur maximale est 255. Modifier la position de ce slider modifie la valeur de l'input $( "#valeur" ) en conséquence.

Cette fonction utilise un bouton radio stylisé $( "#digitalSwitch" ) sous la forme d'un intérrupteur pour "Allumer" ou "Eteindre" un input $( "#ampoule" ).

Cette fonction initialise un spinner $( "#pinValue" ) avec la valeur min à 0, et la valeur max à 255. La valeur de ce spinner est directement liée à la valeur d'un slider $( "#numberSlider" ). La valeur du slider est modifiée à chaque incrémentation ou décrémentation du spinner.
